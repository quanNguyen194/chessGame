# AI Chess Game

## Các bước cài đặt

### Clone

- Clone repo theo đường dẫn https://github.com/quanNguyen194/chessGame.git

### Setup

#### Node.js

- Tải Node.js từ đường dẫn https://nodejs.org/en/download/
- Làm theo các bước được hướng dẫn để cài đặt 

#### Cài đặt khác và khởi chạy

```bash
npm install
npm start
```
